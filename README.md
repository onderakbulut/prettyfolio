## Installation
Clone the project
```
git clone git@github.com:onderakbulut/prettyfolio.git
```
Go to the project directory
```
cd prettyfolio
```
Install dependencies
```
npm install
```
Start the server
```
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
